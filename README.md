#### 项目介绍
ssm+maven+idea开发。
前台页面使用arttemplter模板，bootstrap，jQuery，validform等

该项目的用户主要分为三类：超级管理员（平台管理），管理员（商家管理），用户
超级管理员是平台的管理员，主要是对商家进行操作。
管理员是商家的管理员，主要是对自己商家下的车辆进行管理
用户：浏览平台，租赁车辆时需要登录。在查看车辆和商家信息时会展示商家与自己的位置并进行驾车导航路线规划。
